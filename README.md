
# Arch Installer

A Python-based Arch Linux installer with first-class support for:

- Btrfs
- Snapper
- GRUB
- Reflector
- KDE Plasma
- Automatic hardware detection

## Features

- Automatic CPU detection
- Automatic GPU detection (planned)
- Automatic disk detection
- Interactive installer
- Btrfs subvolumes
- Snapshot support

## Running

```bash
python install.py
```
